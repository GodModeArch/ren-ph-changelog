# REN.PH Changelog

Public audit trail for [ren.ph](https://ren.ph), the Philippine real estate verification platform. 33,504+ verified data nodes covering broker credentials, license-to-sell records, BIR zonal values, and structured property data.

This repository is maintained by the Principal of [godmode.ph](https://godmode.ph). It documents public-facing changes to the platform: new features, data pipeline updates, schema evolution, and infrastructure decisions. The core engine and data pipelines are private. This repo tracks what ships.

See [CHANGELOG.md](CHANGELOG.md) for the full release history.
